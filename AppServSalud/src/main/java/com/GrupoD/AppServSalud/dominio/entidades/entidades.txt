notaciones lombok:

@Data -> es una combinacion de varias notaciones @Getter @Setter @ToString @EqualsAndHashCode @RequiredArgsConstructor
@Getter -> genera los getters
@Setter -> genera los setters
@ToString -> genera el toString
@EqualsAndHashCode -> genera el equals y el hashCode
@RequiredArgsConstructor -> genera un constructor con los atributos marcados como final y los atributos marcados con @NonNull
@NoArgsConstructor -> genera un constructor sin argumentos
@AllArgsConstructor -> genera un constructor con todos los argumentos
